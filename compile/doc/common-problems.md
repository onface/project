## unreleasable file

```shell
[ERROR] unreleasable file [/Users/nimo/Documents/git/project/view/some.js] in [/view/some/index.html]
```

unreleasable file 错误表示你引入的文件不会被产出，请阅读

1. [entry](./entry.md)
2. [vendor](./vendor.md)
3. [ignore](./ignore.md)
4. [unreleasable-file](./unreleasable-file.md)

了解原因。
